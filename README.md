Управление вакансиями с использованием HH API

Этот проект представляет собой консольное приложение для работы с вакансиями с использованием API HeadHunter. Пользователь может выполнять поиск вакансий по различным критериям, просматривать детали о найденных вакансиях и управлять настройками поиска.
Основные функции

    Поиск вакансий:
        Пользователь вводит ключевое слово для поиска вакансий на сайте HeadHunter.
        Программа выполняет запросы к API HH и загружает вакансии, соответствующие запросу.
        Полученные данные сохраняются в файле в формате JSON для последующего использования.

    Фильтрация результатов:
        Пользователь может задать различные фильтры для вакансий:
            Город (фильтрация по местоположению).
            Зарплата (минимальная и максимальная суммы).
            Ключевые слова в описании вакансии.
        После применения фильтров программа отображает только те вакансии, которые соответствуют заданным критериям.

    Вывод результатов:
        Результаты выводятся постранично, с возможностью перемещения между страницами.
        Для каждой вакансии отображаются её основные атрибуты: название, местоположение, зарплата, требования.

    Настройки:
        Пользователь может настроить параметры отображения, такие как количество результатов на странице.

Структура проекта

Проект разделен на несколько основных модулей и классов:

    ViewResults: Основной класс приложения, который управляет всем процессом работы с вакансиями, включая загрузку данных, фильтрацию и вывод результатов.

    Parser: Класс для работы с API HeadHunter. Отвечает за загрузку вакансий по заданному ключевому слову.

    FileWorker: Утилитарный класс для сохранения данных в файл JSON.

    VacancyObject: Класс для представления каждой отдельной вакансии с необходимыми атрибутами (название, местоположение, зарплата, требования).

Использование    

Запуск приложения:
Запустите основной скрипт main.py. Следуйте инструкциям в консоли для выполнения поиска вакансий, установки фильтров и просмотра результатов.

bash

    python main.py

    Настройки:
    Во время работы программы вы можете настраивать параметры поиска и отображения, используя меню настроек.

Примеры использования
Пример 1: Поиск вакансий по ключевому слову

Пользователь вводит ключевое слово "Python developer". Программа загружает все доступные вакансии по этому запросу и сохраняет их в файл.
Пример 2: Фильтрация вакансий по зарплате

После загрузки вакансий пользователь может задать минимальную и максимальную зарплату для фильтрации. Программа отобразит только те вакансии, удовлетворяющие заданным критериям.
Пример 3: Просмотр и переход по страницам результатов

Полученные результаты выводятся постранично. Пользователь может перемещаться между страницами, используя команды в консоли.
